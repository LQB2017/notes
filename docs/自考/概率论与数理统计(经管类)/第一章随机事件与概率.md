# 第一章 随机事件与概率
## 随机事件的关系与运算

事件的包含与相等：
- 事件的包含：事件A发生必然导致事件B发生，记作A⊆B。
- 事件的相等：事件A和事件B同时发生，记作A=B。

和事件：
- 事件A和事件B的和事件，记作A∪B。

积事件：
- 事件A和事件B的积事件，记作A∩B。

差事件：
- 事件A和事件B的差事件，记作A-B。

互不相容：
- 事件A和事件B互不相容，记作A∩B=0。

对立事件：
- 事件A和事件B互为对立事件，记作A=B。

运算律：
- 交换律：A∪B=B∪A，A∩B=B∩A。
- 结合律：(A∪B)∪C=A∪(B∪C)，(A∩B)∩C=A∩(B∩C)。
- 分配律：A∪(B∩C)=(A∪B)∩(A∪C)，A∩(B∪C)=(A∩B)∪(A∩C)。
- 德摩根律：(A∪B)C=AC∩BC，(A∩B)C=AC∪BC。
- 对偶律：A∪B=A∩B，A∩B=A∪B。

## 概率

古典概型：
- 古典概型的特点：
  - 试验的样本空间是有限的(解释：样本空间中的元素个数是有限的)。
  - 试验的每个结果是等可能的(解释：每个结果出现的概率相等)。
- 古典概型的概率计算公式：
  - P(A)=m/n，其中m是事件A的样本点数，n是样本空间的总样本点数。

概率的性质：
- 性质1：对于任一事件A，有0<=P(A)<=1 （解释：概率不能为负，也不能大于1）。
- 性质2 加法公式：对于任一事件A和事件B，有P(A∪B)=P(A)+P(B)-P(A∩B) （解释：事件A和事件B的并集的概率等于事件A的概率加上事件B的概率减去事件A和事件B的交集的概率）。
- 性质3 减法公式：对于任一事件A和事件B，有P(A-B)=P(A)-P(A∩B) （解释：事件A的概率等于事件A的概率减去事件A和事件B的交集的概率）。
- 性质4 对立事件公式：对于任一事件A，有P(A)=1 - P(A) （解释：事件A的概率等于1减去事件A的补集的概率）。

几何概型：
- 几何概型的特点：
  - 试验的样本空间是无限的。
  - 试验的每个结果是等可能的。
